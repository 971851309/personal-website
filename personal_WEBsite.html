Research Projects

Multi-modal Tourism Recommendation System with LLM (MMAgentRec) 2023.04 - 2024.05

  • Built a high-quality tourism dataset with 9,004 entries from Guangdong province, using BERT for text and ResNet50 for image processing
  • Designed a Multi-Graph Convolution Network (MGCN) framework with cross-attention mechanism for multi-modal fusion
  • Integrated self-reflective LLM for continuous recommendation optimization through user feedback
  • Outperformed Blip2 and CLIP on accuracy, recall and F1 scores
  • Published in SCI Q2 journal (sole author, accepted)
  • DOI: 10.1038/s41598-025-96458-w

DCNN-BERT-BiLSTM Multi-modal Sentiment Analysis Framework (MEA-IFE) 2023.01 - 2024.01

  • Developed BERT-SKNet-BiLSTM-ECA text feature extraction module with enhanced multi-scale context modeling
  • Designed image feature enhancement combining AlexNet and Vision Transformer
  • Implemented multi-head attention for dynamic feature fusion, improving cross-modal interaction
  • Achieved 98.00% accuracy and 98.13% F1 score on Kaggle text and FER2013 image datasets
  • Published in SCI Q3 journal (second author)

Medical Image Visual Question Answering (VQA) System 2024.04 - 2024.10

  • Collected and processed 5,640 medical images with 46,827 QA pairs (X-rays, CT scans, pathology slides)
  • Designed multi-modal VQA framework with MGCN, ResNet and cross-attention mechanisms
  • Achieved 0.9150 accuracy, 0.93 recall and 0.92 F1 score on MedVQA dataset
  • Submitted to TPAMI (CCF-A, under review)

Poppy Recognition System for Anti-drug Drones 2019.11 - 2021.10

  • Led neural network algorithm development using TensorFlow 2.0 and Python
  • Collected and labeled 2,000 poppy images; achieved 70% recognition accuracy
  • Implemented in collaboration with Public Security Department
  • National Innovation Project (approved)

Bitcoin-Gold Portfolio Investment with Deep Reinforcement Learning 2022.01 - 2022.06

  • Developed PPO algorithm for quantitative trading decisions
  • Used LSTM expectations as input to DRL, improving accuracy by 10.38%
  • Published conference paper
